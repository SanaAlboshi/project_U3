# Cultural Institutions & Museums Management

This project is a Django web application for managing cultural institutions and their associated museums. Users can browse institutions and museums, view details, and make reservations.

## Features

- Add and manage cultural institutions.
- Each institution can manage its own museums.
- Users can browse institutions and museums.
- Users can view museum details, including opening hours and description.
- Users can make reservations for museums.
- Admin panel to manage institutions, museums, and bookings.
